The attached zip contains the code and the cors policy (cors.json) and bucket policy(BucketPolicy.json) for S3 bucket. It also contains Lambda function
code and the code for the front end of the website. 

The template to create the bucket is Stack.json
The code for the front end is index.html
The code for the Lambda function to get the signed URL is signedURL_Lambda.js
The code for the Lambda function to insert to DynamDB is DynamoInsertLambda.js
